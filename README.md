# 香港收銀車服務地圖 (HK Coin Collection Map) 🪙🚛

這是一個全自動化的開源項目，旨在將香港金融管理局（HKMA）的「收銀車」服務日程表視覺化。系統每日自動從官方來源擷取最新數據，並透過靜態網頁地圖展示當日服務位置。

## 🌐 項目特點

- **全自動更新**：每日早上 10:00 (HKT) 透過 GitHub Actions 自動執行 Python 腳本，擷取最新 PDF 並更新數據。
- **互動式地圖**：使用 Leaflet.js 渲染香港十八區邊界，直觀展示收銀車 1 號及 2 號的精確區域。
- **自動化數據清洗**：內建強大的 PDF 解析器，能處理金管局日程表中複雜的日期與地點分拆邏輯。
- **無後端架構**：完全基於 GitHub Pages 部署，無需伺服器維護。

## 📊 數據來源

1. **服務日程**：[金管局收銀車日程表 (PDF)](https://www.hkma.gov.hk/media/chi/doc/key-functions/monetary-stability/notes-and-coins/chi_coin_collection.pdf)
2. **十八區邊界**：[地政總署 CSDI 地理空間數據庫 (WFS)](https://portal.csdi.gov.hk/)

## ❓ 常見問題 (FAQ)

### 1. 數據多久更新一次？
系統設定於香港時間**每日上午 10:00** 自動執行。如果金管局更新了 PDF，地圖通常會在 15 分鐘內反映最新改動。

### 2. 如果遇到惡劣天氣（如黑雨或八號風球），地圖會顯示嗎？
金管局通常會在 PDF 備註中列出預定的暫停服務日，系統會自動抓取並顯示為「今日暫停」。但對於突發性的天氣暫停，建議仍以金管局官網即時公告為準。

### 3. 我可以手動觸發更新嗎？
可以。在 GitHub Repo 的 **Actions** 頁籤中，點擊 "Update data and deploy Pages" 並選擇 "Run workflow" 即可手動重新抓取資料。

### 4. 為什麼地圖上有些區域是深色的？
- 🟡 **黃色**：收銀車 1 號所在區域
- 🔵 **藍色**：收銀車 2 號所在區域
- 🟣 **紫色**：兩架車同時在該區域
- 🔴 **紅色**：該區今日預定暫停服務
- 🌑 **深色**：今日無服務
